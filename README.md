# Electron Planner
A simple electron-powered organizer and planner for personal use! Features include notes creation, editing, deletion, as well as tagging, searching, and pinning.

## Credits
Base code and implementation provided by [ITowerEducation](https://www.youtube.com/@ITowerEducation) - the tutorial was used to set up the initial structure quickly. Basic setup steps were skipped, as the focus was on implementing custom features. 

All other modifications, beyond what is shown in the tutorial were implemented by me.
